# ForgeEd Presentation Website

## Overview

This is a static presentation website for the ForgeEd AI-Enhanced Learning Management System project. It showcases the project's executive summary, problem context, proposed solution, architecture, database design, features, and team information.

## Project Structure

```
/
├── index.html           # Main HTML file with all content sections
├── styles.css           # CSS styles with ForgeEd brand colors (blue/orange)
├── script.js            # JavaScript for navigation and animations
├── README.md            # GitHub Pages hosting instructions
├── replit.md            # This file
└── assets/
    └── images/
        ├── logo.png           # ForgeEd logo
        ├── team.jpeg          # Team photo
        ├── architecture.png   # System architecture diagram
        └── er-diagram.png     # Database ER diagram
```

## Purpose

This website is designed to:
1. Present the ForgeEd project in a professional format
2. Be hosted on GitHub Pages as a static site
3. Provide navigation to different sections of the presentation
4. Display all project assets (logo, team photo, diagrams)

## Technology

- Pure HTML5, CSS3, and vanilla JavaScript
- No build process required
- Google Fonts (Inter) for typography
- Font Awesome for icons
- Responsive design for all screen sizes

## Running Locally

The website is served using Python's built-in HTTP server on port 5000.
